# Financial67_Backend

## User Route

List of User routes are :
<br>
POST `http://localhost:3001/register` <br>
POST `http://localhost:3001/login` <br>
GET `http://localhost:3001/users` <br>
GET `http://localhost:3001/user/:id` <br>

Payload is : 
``` JSON 
{
    "name":"mohit",
    "email":"123@abc.com",
    "password":"!@#1",
    "usertype":"student"
}
```
